# 【設定系（Set 系）メソッドまとめ】

## 用途

- 値や状態を設定・代入する
- 単純代入から複雑な初期化・設定まで

## 命名ルール

- 動詞＋目的語で命名
- 単純代入は `Set`、複雑な設定や初期化は `Apply` / `Configure` を使用

## 命名例

| メソッド | 説明 |
| --- | --- |
| `SetVolume()` | 音量を設定する |
| `ApplySettings()` | 設定を適用する |
| `AssignRole()` | 役割を割り当てる |
| `ConfigureDatabase()` | データベースを構成する |

## 注意点

1. 値の設定や副作用があることを名前で明示する
2. 設定対象が明確になるよう命名する
3. 初期化系（`Init` / `Setup`）との混同に注意
