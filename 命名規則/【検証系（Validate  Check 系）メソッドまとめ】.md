# 【検証系（Validate / Check 系）メソッドまとめ】

## 用途

- 条件や値の検証、整合性チェック
- 主に `bool` を返す

## 命名ルール

- 動詞＋目的語で命名
- `Ensure` は例外を投げる場合に使用

## 命名例

| メソッド | 説明 |
| --- | --- |
| `ValidateInput()` | 入力値を検証する |
| `CheckPermission()` | 権限を確認する |
| `VerifyData()` | データを検証する |
| `EnsureInitialized()` | 初期化済みか検証する |
| `IsValid()` | 有効か判定する |

## 注意点

1. 戻り値は `bool` が基本
2. 副作用は原則避ける
3. 複雑な条件の場合はメソッド名で意味を明確化
4. `Ensure` 系は例外を投げる場合に限定
