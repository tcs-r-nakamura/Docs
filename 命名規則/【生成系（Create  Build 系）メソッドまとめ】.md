# 【生成系（Create / Build 系）メソッドまとめ】

## 用途

- オブジェクトやデータを新規生成する
- 副作用の有無を明確にする

## 命名ルール

- 動詞＋目的語で命名
- 副作用がある場合は名前に明示

## 命名例

| メソッド | 説明 |
| --- | --- |
| `CreatePlayer()` | プレイヤーを生成する |
| `BuildReport()` | レポートを作成する |
| `GenerateId()` | ID を生成する |
| `InstantiateEnemy()` | 敵オブジェクトを生成する |
| `MakeCopy()` | コピーを作成する |

## 注意点

1. 生成後の副作用（DB登録、ファイル作成など）があれば名前で明示
2. `Create`/`Build` は更新系や読み込み系と混同しない
3. 戻り値で生成物を返すことが基本
