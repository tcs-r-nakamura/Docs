# 【変換系（Convert / To 系）メソッドまとめ】

## 用途

- データ型や形式を変換する
- 型変換、DTO 変換、座標変換など

## 命名ルール

- 動詞＋目的語で命名
- 変換先や意味を明示

## 命名例

| メソッド | 説明 |
| --- | --- |
| `ConvertToJson()` | JSON に変換する |
| `ToString()` | 文字列に変換する |
| `ParseInt()` | 整数に変換する |
| `MapToDto()` | DTO にマッピングする |
| `TransformCoordinates()` | 座標を変換する |

## 注意点

1. 元のデータは変更せず、新しい形式で返すことが基本
2. 副作用がある場合は名前で明示
3. 変換先の型や形式が分かるように命名する
