# 【並列・非同期系（Async 系）メソッドまとめ】

## 用途

- 非同期処理、時間のかかる処理
- `Task` / `Task<T>` を返す

## 命名ルール

- メソッド名に必ず `Async` を付与
- 呼び出し側で `await` が使えることを明示

## 命名例

| メソッド | 説明 |
| --- | --- |
| `LoadDataAsync()` | データを非同期で読み込む |
| `SaveFileAsync()` | ファイルを非同期で保存 |
| `FetchUserDataAsync()` | ユーザデータを非同期取得 |
| `ProcessTasksAsync()` | タスクを非同期で処理 |

## 注意点

1. 戻り値は `Task` / `Task<T>` で統一
2. 呼び出し側で `await` 可能であることを明示
3. `Async` メソッドは内部で例外管理やキャンセルを考慮
4. 長時間処理や副作用を伴う場合に使用
