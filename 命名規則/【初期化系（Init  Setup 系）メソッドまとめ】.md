# 【初期化系（Init / Setup 系）メソッドまとめ】

## 用途

- オブジェクトや状態の初期化処理
- 状態リセットも含む

## 命名ルール

- 動詞＋目的語で命名
- `Reset` は状態を元に戻すことを意図

## 命名例

| メソッド | 説明 |
| --- | --- |
| `InitializeComponents()` | コンポーネント初期化 |
| `SetupScene()` | シーンをセットアップ |
| `ResetGame()` | ゲーム状態をリセット |
| `PrepareData()` | データ準備 |

## 注意点

1. 初期化範囲や影響範囲を明確にする
2. `Reset` は元に戻す意味で限定的に使用
3. 副作用がある場合は名前に反映
