# 【イベント・通知系（On / Notify / Raise / Trigger 系）メソッドまとめ】

## 用途

- イベントの発火や通知処理

## 命名ルール

- イベントハンドラ：`OnXxx`
- 通知処理：`Notify` / `Raise` / `Trigger` を使用
- 副作用あり

## 命名例

| メソッド | 説明 |
| --- | --- |
| `OnClick()` | クリックイベント発火 |
| `NotifyUser()` | ユーザに通知 |
| `RaiseEvent()` | イベントを発火 |
| `TriggerAnimation()` | アニメーションを開始 |

## 注意点

1. メソッドが副作用を伴うことを名前で明示
2. イベントハンドラは `OnXxx` に統一
3. 非同期イベント処理は `Async` を検討
