# ✔ using は「必ず Dispose() を呼んで後片付けをする」ための構文

最も重要な役割はこれです：

リソース（ファイル、DB、ネットワーク等）を自動で解放するための仕組み。`finally` で `Dispose()` を呼ぶのと同じ働きをする。外部リソースを解放するための仕組み。

---

## GC が回収するもの

.NET の マネージドメモリ（`new` したオブジェクト）

## GC が回収できないもの

OS が管理する "外部リソース"（アンマネージドリソース）

- ファイルハンドル
- Win32 ハンドル（HFILE、HANDLE）
- DB 接続
- ソケット
- COM オブジェクト
- デバイスドライバのリソース
- GDIオブジェクト（Bitmap, Graphics など）

➡ .NET の GC は上記を知らないので解放できない。
