# Transaction

| 用語 | 意味 |
| -------------------- | ------------------------- |
| `BeginTransaction()` | トランザクション（まとめて処理）を開始する |
| `Commit()` | 成功したので処理を確定 |
| `Rollback()` | エラーが出たので全部なかったことにする（巻き戻し） |

| 用語 | 意味 |
| ------------------------- | --------------------------- |
| **Transaction**（トランザクション） | 「すべて成功 or すべて失敗」でまとめて処理する単位 |
| **Commit** | 成功として確定する |
| **Rollback** | 失敗として元に戻す |

---

## ExecuteReader()

| メソッド | 戻り値 | 役割 |
| ----------------- | --------------- | --------------------------- |
| `ExecuteReader()` | `SqlDataReader` | SQLのSELECT文を実行し、結果を行単位で読み取る |

| ポイント | 内容 |
| -------------------------------------------------- | -- |
| ✔ `ExecuteReader()` は標準で使えるメソッドです。 | |
| ✔ ただし、**接続を開いた状態で使う**必要があります。 | |
| ✔ 使う前に `SqlCommand` と `SqlConnection` を正しく準備しましょう。 | |

---

## SqlConnection とは？

SQL Server に接続するためのクラス（他のDBでは使えません）
