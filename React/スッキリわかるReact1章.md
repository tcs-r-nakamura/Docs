# React 1章

## 用語集

**Node.js（ノード・ジェイエス）**
JavaScriptでサーバーやツールを作れるようにしたもの

**npm（エヌピーエム）**
JavaScriptのライブラリを「入れる・消す・更新する」を管理する道具

**pnpm（ピーエヌピーエム）**
npm の仲間で、より高速・軽量にライブラリを管理できるツール

**Visual Studio Code（VS Code）**
プログラムを書くための超便利なメモ帳（型の整合性チェックや Null 安全性のチェックとかをコーディング中に自動でやってくれる）

**Vite（ヴィート）**
すぐ作れてすぐ確認できる、超高速Web開発ツール

**CSS（Cascading Style Sheets）**
Webページの見た目（デザイン）を指定するための言語

**コンポーネント（Component）**
ボタンやヘッダー、カードなどの「使い回せるパーツ」

**スクリプト（script）**
コンピュータにやらせる指示書
